1. Identify dataset(s)
Identify the dataset(s) you would like to get access to.
2. Apply for access
Identify the Data Access Committee (DAC) affiliated with the dataset(s) you are interested in. Then, contact the DAC to start your data access application.
3. Receive download instructions
Once the Data Access Committe (DAC) has accepted your data access request, you will recieve instructions from the SDA Helpdesk on how to access the data.
4. Download and decrypt files
Download the files and decrypt them by following the instructions provided by the SDA Helpdesk.
