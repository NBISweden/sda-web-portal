1. Identify dataset(s)
Identify the dataset(s) you want to request access for.
2. Request access
Identify the Data Access Committee (DAC) affiliated with the dataset(s) you wish to request access for. Then, contact the DAC to start your data access application. A Data Access Agreement (DAA) documenting the terms of access and usage of the dataset needs to be negotiated between you and the DAC and should be signed by both parts.
3. Receive download instructions
Once the DAC has accepted your data access request and an authorisation grant has been received by the SDA Helpdesk, you will recieve instructions from the SDA Helpdesk on how to access the data.
4. Download and decrypt files
Follow the SDA-specific instructions on GitHub on how to download and decrypt files. Please note that you need to confirm successful download and decryption back to the SDA helpdesk.
